# videodownloader
A Simple Script writtten in Python to Download YouTube Videos
These videos can also be downloaded as audio

### Installation
1. Clone the Repo
```Bash
cd videodownloader
# run pipenv shell
pipenv shell
# Then install the dependencies
pipenv install
# To Run the script 
python3 script.py
```
